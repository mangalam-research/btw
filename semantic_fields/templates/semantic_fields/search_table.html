Type the text you want to search for in this field. If you surround the text with double quotes, the search will be "exact". For instance, if you search for "Speak", it will match only the word Speak, not Speaking, or speak.

This selector determines what aspect of the semantic fields you want to search: semantic field names will search only the names of the semantic fields, lexemes will search only the lexemes associated with the fields.

This selector determines what the scope of the search is. Use HTE fields to select only those fields provided with the Historical Thesaurus of English. Use custom BTW fields to search those fields created for the BTW project. Use all fields to search all fields.

This option restricts the set of semantic fields to those who are descendants of the option selected. The all fields option means no restriction is performed.
